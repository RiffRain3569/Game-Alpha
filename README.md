# Game-Alpha (0-a 프로토타입)

2D 탑다운 근접 교전 프로토타입 — Godot 4.x / GDScript

## 검증 목표

"위치로 각 잡는 재미 + 각목으로 실행하는 손맛"이 반응속도 게임으로 무너지지 않고 공존하는지 확인.

## 기술 스택

- Godot 4.7 stable
- GDScript
- 2D 렌더러

## 실행 방법

1. Godot 4.7을 설치
2. 이 폴더를 Godot에서 프로젝트로 열기
3. F5로 실행
